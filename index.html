Gaming Projects

Impulse Instagib (Current Project)

Unreal Engine 4 multiplayer game that focuses on rocket jumping, bunny hopping, and other skill based movement for a fast paced first person shooter. Based on the official multiplayer tutorial from the UnrealEngine YouTube channel.

Rolling Ball Game

Fully fledged Unity3D rolling ball game with instructions, a tutorial, simple enemy AI, 3 difficulty settings with 3 separate levels, and other features. Play by launching Release2.exe in the builds folder.

Patient Zero

Large Unity3D first person project based on the previous repository named COMP305-Final-Project. Unlike the previous project, in this game you're patient zero and you try to convert humans to survive while they run from you. More information in the design and external documents.

Pathfinding Experiment

Unity3D school assignment where you can select shapes by left clicking on them, and right click to tell the group where to go. The yellow color of the shapes indicated which one is selected. Each shape will use path finding to navigate around structures to the destination.
